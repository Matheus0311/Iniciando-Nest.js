##Estou iniciando meus estudos com Node.js utilizando o framework Nest.js

## Instalação

```bash
$ npm install
```

## Rodando o app

```bash
# desenvolvimento
$ npm run start

# modo watch
$ npm run start:dev

# modo produção
$ npm run start:prod
```

## Test

```bash
# testes unitários
$ npm run test

# testes e2e
$ npm run test:e2e

# cobertura de teste
$ npm run test:cov

```
